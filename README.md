### Method 2: Run from Source
1. Clone this repository:
```bash

```

2. Create and activate a virtual environment (recommended):
```bash
python -m venv .venv
.venv\Scripts\activate  # On Windows
```

3. Install the required dependencies:
```bash
pip install pygame
```

4. Run the game:
```bash
python Main/game/__main__.py
```
